# resilient-blackout

Clean-room grid physical risk model for resilient infrastructure analysis.

## Status

**Research library — not deployed to production.**

This package models grid blackout risk, compound climate hazards, and investment
planning for distribution networks. It is not referenced by
[`resilient-backend`](https://github.com/Resilient-World/resilient-backend) or
the production model mesh.

## Install

```bash
pip install -e ".[dev]"
```
